Criação e uso de e-mail

Passo 1: Escolhendo um serviço de e-mail

Existem vários serviços de e-mail gratuitos. Os mais populares são Gmail (Google), Outlook (Microsoft) e Yahoo Mail. Para este tutorial, vamos usar o Gmail.

Passo 2: Criando sua conta

1. Acesse o site gmail.com no seu navegador
2. Clique em "Criar conta"
3. Preencha o formulário com seus dados pessoais
4. Escolha um nome de usuário e senha segura
5. Aceite os termos de serviço e clique em "Próxima"

Passo 3: Enviando seu primeiro e-mail

1. Após fazer login, clique em "Escrever" no canto superior esquerdo
2. No campo "Para", digite o endereço de e-mail do destinatário
3. Preencha o assunto e escreva sua mensagem
4. Clique em "Enviar"

Passo 4: Organizando sua caixa de entrada

Você pode organizar seus e-mails criando pastas (chamadas de "rótulos" no Gmail):
1. À esquerda da tela, clique em "Mais" e depois "Criar novo rótulo"
2. Dê um nome para a pasta (ex: "Família", "Trabalho")
3. Para mover um e-mail para uma pasta, arraste-o para o rótulo desejado

Navegação por Teclado

Use estas teclas para navegar no site:

  • Tab - Mover para o próximo elemento
  • Shift + Tab - Mover para o elemento anterior
  • Enter - Ativar botão ou link
  • Espaço - Marcar/desmarcar caixa de seleção
  • Setas - Navegar entre opções
  • Esc - Fechar modal ou menu

Para navegar entre os módulos, use a tecla Tab até chegar no módulo desejado e pressione Enter para abrir.
